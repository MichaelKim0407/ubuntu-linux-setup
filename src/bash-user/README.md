# Bash (user)

Per-user bash configuration — PS1, aliases, functions, and bashrc customizations.

Installs `~/.bash_append` (symlink) and `~/.bash_ps1` (copy), and appends
`source ~/.bash_append` to `~/.bashrc`.

Run as root from the repo root to install for the root user:

```bash
install/bash-user.sh
```

Run as the non-root user to install for that user:

```bash
install/bash-user.sh
```
